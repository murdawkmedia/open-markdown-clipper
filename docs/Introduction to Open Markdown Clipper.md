Open Markdown Clipper is an independent browser extension for highlighting pages and capturing web content as durable Markdown.

> [!warning] Development build
> Version 0.1.0 is not published in a browser store. Build and load it from this repository while pre-release verification is in progress.

## Install Web Clipper

During development, build Open Markdown Clipper locally and load the package for your browser:

- Chromium browsers: load the unpacked `dist/` directory from `chrome://extensions`.
- Firefox: load `dist_firefox/manifest.json` from `about:debugging#/runtime/this-firefox`.

See the repository [README](../README.md) for build and verification commands. Safari is not packaged.

## Privacy

Open Markdown Clipper has no telemetry. Copy and Download stay in the browser; optional destinations transmit content only when the user activates them. The code is [open source](https://github.com/murdawkmedia/open-markdown-clipper) and auditable.
