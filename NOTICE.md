# Open Markdown Clipper notice

Open Markdown Clipper is an independent fork of Obsidian Web Clipper:

https://github.com/obsidianmd/obsidian-clipper

The fork was created on 2026-07-12. The upstream source is provided under the
MIT License; its copyright and license notice are preserved in `LICENSE`.

Material changes include an independent product and package identity,
Firefox-and-Chromium-only packaging, original replacement artwork, removal of
the application-specific transport and browser-based AI interpretation setup,
and generic Copy, Download, Custom URI, and authenticated loopback HTTP
destinations for captured Markdown. The fork also adds destination-isolated
credential storage, Firefox optional data consent, Quick Clip destination
routing, and public-release privacy checks.

Open Markdown Clipper is independent and is not affiliated with, sponsored by,
or endorsed by Obsidian.
